Abrir chatbot del curso

¿Qué te llevas de esta sesión?

  • Plantillas de prompts para redactar y revisar textos académicos.
  • Un flujo de trabajo para revisión bibliográfica asistida por IA.
  • Buenas prácticas para evitar alucinaciones y errores de citación.
  • Checklist de uso ético y responsable en el contexto universitario.

Programa sugerido (120 min)

  1. Fundamentos útiles de IA generativa (15 min)
  2. Prompting aplicado al trabajo académico (30 min)
  3. Taller: revisión bibliográfica y síntesis (30 min)
  4. Automatizaciones sencillas (20 min)
  5. Riesgos, ética y evaluación crítica (15 min)
  6. Q&A + próximos pasos (10 min)

Recursos que puedes preparar ya

  • Un artículo reciente para practicar resumen crítico.
  • Un borrador (introducción o discusión) para reescritura guiada.
  • Una tabla de resultados para generar interpretaciones alternativas.
  • Una rúbrica de evaluación para convertirla en prompt de feedback.

Despliegue recomendado

Para este proyecto (web estática + endpoint de chatbot con claves privadas), la opción más cómoda es Vercel o Render:

  • Vercel: despliegue muy rápido desde GitHub, variables de entorno seguras y buena experiencia para páginas de curso.
  • Render: sencillo para mantener un servidor Node.js 24/7 con endpoint de chat.

Importante: no pongas nunca las API keys en el frontend. Este proyecto ya está preparado para usarlas en servidor.
